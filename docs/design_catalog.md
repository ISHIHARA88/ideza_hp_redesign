# 💎 Ultra-Atmospheric Design Catalog

石原さんのクリエイティブを劇的に進化させる、世界観を構築するための「技術の引き出し」です。

## 1. 極限の「大気」ぼかし (Extreme Blur)
*   **技術**: `backdrop-filter: blur(160px);`
*   **効果**: 通常（10-20px）の数倍のぼかしをかけることで、コンテンツが背景に「置かれている」のではなく、光の中に「滞留している」感覚を生みます。

## 2. 真珠光沢グラス (Pearlescent Glass)
*   **技術**: `linear-gradient` と `backdrop-filter` の組み合わせ。
*   **効果**: 枠線（Border）を使わず、わずかに色味の違うグラデーションを重ねることで、真珠のような上品な光沢を表現します。

## 3. リキッド・シャンパン (Molten Metal Gradient)
*   **技術**: 金・ブロンズ・白の不規則な階層グラデーション。
*   **効果**: 溶けた金属のような高級感をボタンやアクセントに付与します。

## 4. ミスト・エッジ (Mist-Edge Masking)
*   **内容**: CSS `mask-image` を使用した端の透過。
*   **効果**: セクションの境界を煙のように消し去ります。画像上のテキストが見えにくい場合に、背景の一部だけを霧で覆うように配置できます。

## 5. シネマ・スケール余白 (Storytelling Spacing)
*   **内容**: セクション間に 320px〜400px の余白を設置。
*   **効果**: スクロール中に「何もない豊かな時間（間）」を演出。ハイブランドの店舗内のような、一歩一歩を楽しむ体験に変わります。

## 6. スナッピー・ズーム (0.3s Snap Interaction)
*   **技術**: `transition: 0.3s` と `transform: scale()`。
*   **効果**: 触れた瞬間に「スッ」と動く応答性の良さは、高級品を扱う際の心地よさを感じさせます。
