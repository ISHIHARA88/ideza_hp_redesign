# 🏺 ブティック・Web制作サービス運用ガイド

石原さんが「カスタムコード制作」をパッケージ商品として納品するための運用手引書です。お客様への説明にそのままお使いいただけます。

## 1. アカウント登録の手順
この最高級のサイトを公開・運営するために、以下の2つのサービスのアカウント登録をお客様にお願いします。

### ① Netlify（ネットリファイ）- ホスティングサービス
*   **用途**: Webサイトを世界中に表示するための「土地（サーバー）」です。
*   **手順**: 
    1. [Netlify.com](https://www.netlify.com/) で「Sign Up」
    2. 石原さんが制作した完成フォルダを、そこのダッシュボードに「ドラッグ＆ドロップ」するだけで公開完了です。
*   **費用**: 基本無料（アクセスが非常に多い場合のみ有料プラン）。

### ② Formspree（フォームスプリー）- フォーム連携サービス
*   **用途**: お問い合わせが石原さんやお客様のメールに届くようにする「郵便局」です。
*   **手順**:
    1. [Formspree.io](https://formspree.io/) でアカウント作成
    2. 私（AI）が作成した「フォーム用のアドレス（Endpoint URL）」をコードに1箇所貼るだけで連携完了です。
*   **費用**: 無料（月50通以上届く場合は有料プラン）。

## 2. ドメインの設定方法
「ideza.jp」などの独自ドメインを使いたい場合の説明です。
1.  お客様に「お名前.com」や「ムームードメイン」でドメインを取得してもらいます。
2.  Netlifyの管理画面から、表示される「DNSレコード（英数字）」をコピー。
3.  ドメイン取得サービスの管理画面に、その英数字を貼り付けます。
*   **お客様への一言**: 「設定は複雑に見えますが、私がサポートしますので10分ほどで終わります」とお伝えください。

## 3. メールアドレスの連携
お問い合わせの内容をどこに飛ばすかの設定です。
1.  Formspreeのダッシュボードで、「Notification Email」の欄にご希望のメールアドレスを入力。
2.  届いた確認メールのボタンを押す。
*   これで、「お客様のサイトからメールが届いた！」という通知が即座に飛ぶようになります。
